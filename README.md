# QR App

A modern and clean React Native application for **scanning** and **generating** QR codes.  
Built with **Expo**, featuring fast performance and a simple UI.

## Features
- Scan QR codes using the device camera
- Generate custom QR codes
- Minimal and clean interface
- Works on both iOS and Android

## Tech Stack
- React Native
- Expo Router
- react-native-qrcode-svg
